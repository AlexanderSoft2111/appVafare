## Tecnologías y lenguaje
- **Angular 20** (TypeScript)
- **Ionic 8** (UI components)
- **HTML / SCSS**
- **Firebase** (Auth, Firestore, Storage, Hosting)

## Requisitos
- Node.js LTS y npm
- Ionic CLI (`npm i -g @ionic/cli`)
- Cuenta y proyecto en Firebase

## Configuración Impresora Epson TM-M30II
1. Coloca una direccion IP Fija a la impresora ejemplo: 192.168.1.50
2. Genera el cerficado de confianza.
3. Importa el certificado en opciones de internet en confianza de certificados raiz.

## Configuración Impresora Zebra ZD888
1. Instala el driver ZDDriver de la pagina oficial de zebra.
2. Configura las preferencias de impresora:
    - Tamano de papel ancho y alto
    - Modo de impresion
    - Opciones de graficos
    - Calibrar
3. Instalar Browser print en la pc, descargar de la pagina oficial de zebra
